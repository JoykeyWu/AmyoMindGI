# 🌾 AmyoMind · GI智能体

> 升糖指数（Glycemic Index）科普与低GI产品研发专属AI顾问

## 简介

AmyoMind 是一款专注于**升糖指数（GI）**领域的垂直智能体，依托专业知识库与 DeepSeek 大模型，为营养师、食品研发人员及健康管理爱好者提供：

- 🔬 GI/GL 原理与测定方法科普
- 🧪 低GI产品配方与工艺研发建议
- 🌿 抗性淀粉、膳食纤维、淀粉回生机制解析
- 🩺 血糖管理与糖尿病饮食指导

## 使用方法

1. 打开链接：[https://joykeywu.github.io/AmyoMindGI/]https://joykeywu.github.io/AmyoMindGI/
2. 在页面底部填入你的 [DeepSeek API Key](https://platform.deepseek.com)
3. 点击保存后即可开始对话（Key 仅存储在本地浏览器，不会上传）
